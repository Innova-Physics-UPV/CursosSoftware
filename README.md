# CursosSoftware
Pequeños temarios y ejercicios para nuevos integrantes
## Descarga y utilización
La descarga se puede realizar descargando el .zip desde la ventana code.
Para un aprendizaje más prolifico es interesante y necesario que se descarge utilizando git (clone).
Para ello dejaremos aqui multiples referencias cortas, siempre se recomienda revisar todas para obtener una imagen algo más general.
Git es simplemente un gestor de versiones, es fácil fijarse en que da nombre a la plataforma para alojar proyectos github.
Git es un software que se puede controlar simplemete con comandos desde consola , puedes instalar esta versión aquí:(windows) https://gitforwindows.org/, (otros)https://git-scm.com/install 
En cualquier caso, para principiantes es interesante instalar una version con interfaz gráfica porque nos muestra que menus y opciones tenemos disponibles sin tener que acordarnos de los nombres de estas ni saber si se pueden ejecutar o no, se puede desgargar una aquí : https://git-scm.com/tools/guis. Si no sabes cual instalar, github desktop es una buena opción.
Una vez tienes git instalado deberás clonar el repositorio, hay numerosos tutoriales explicando como hacer esto, así que no lo explicaré.
Si tienes github desktop puedes usar este artículo:https://docs.github.com/es/desktop/adding-and-cloning-repositories/cloning-a-repository-from-github-to-github-desktop
Enlazo también el articulo de github al respecto aunque no lo he encontrado especialmente orientativo. https://docs.github.com/es/repositories/creating-and-managing-repositories/cloning-a-repository. 
Este es algo más orientativo: https://4geeks.com/es/how-to/como-clonar-un-repositorio-de-github
Desde la mayoría de IDEs tambien puedes clonar repositorios.
